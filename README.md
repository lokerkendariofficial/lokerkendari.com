## 📖 Tentang Proyek

**Loker Kendari** adalah platform informasi lowongan kerja yang dirancang khusus untuk masyarakat Kendari dan Sulawesi Tenggara. Website ini menyajikan informasi lowongan kerja dari berbagai sumber terpercaya secara transparan dan gratis.

Proyek ini dibangun dengan arsitektur modular menggunakan **HTML**, **CSS**, dan **JavaScript (ES Modules)** tanpa framework berat, sehingga ringan dan cepat diakses.

---

## 🎯 Fitur Utama

| Fitur | Keterangan |
|-------|------------|
| ✅ **Daftar Lowongan** | Menampilkan lowongan kerja dalam bentuk kartu dengan informasi lengkap |
| ✅ **Filter & Search** | Cari lowongan berdasarkan kata kunci, kategori, dan urutan |
| ✅ **Theme System** | 4 tema siap pakai (Light, Dark, Modern, Aqua) |
| ✅ **Upload Informasi** | Form untuk mengirimkan informasi lowongan (HRD/Admin) |
| ✅ **Dashboard Monitor** | Pantau status file dan performa website |
| ✅ **Responsive Design** | Tampilan optimal di desktop, tablet, dan mobile |
| ✅ **Error Handler** | Sistem pelacakan error terintegrasi |
| ✅ **Design System** | Komponen UI konsisten dengan sistem desain terpisah |
