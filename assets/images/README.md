# Folder Images

Folder ini berisi logo dan favicon untuk website Loker Kendari.

## File Vektor (Sumber)

| File | Deskripsi |
|------|-----------|
| `logo.svg` | Logo utama (horizontal) untuk header, footer |
| `logo-icon.svg` | Logo ikon persegi untuk monitor, upload |
| `favicon.svg` | Favicon untuk tab browser (vektor) |

## File Hasil Konversi

| File | Deskripsi | Ukuran |
|------|-----------|--------|
| `logo.webp` | Logo utama (format modern) | 400×100 |
| `logo-icon.webp` | Logo ikon persegi | 200×200 |
| `favicon.ico` | Favicon untuk browser | 64×64 |
| `favicon.png` | Favicon fallback | 64×64 |

## Cara Menggunakan di HTML

### Logo di Header
```html
<img src="assets/images/logo.webp" alt="Loker Kendari" width="200" height="50" />
